JavaScript and Java

Despite similar names, JavaScript and Java are distinct languages.

JavaScript is interpreted, while Java is compiled.

  • Different use cases
  • Different syntax

Hello world

Most programming tutorials start with a "Hello, world!" example.

It's a simple way to show the syntax and output process.

alert("Hello, world!");

Variables

Variables store data values in JavaScript.

You can declare variables using let, const, or var.

  • let for block-scoped variables
  • const for constants
  • var for function-scoped variables (older)
